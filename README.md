## ✨ Реализованный функционал

### 🔐 Авторизация
- Вход в систему через получение Bearer-токена (Laravel Sanctum)
- Сохранение сессии в `localStorage`
- Защита приватных маршрутов и элементов интерфейса

### 📂 Проекты
- **Список проектов:** загрузка данных с API, карточки с рейтингом, тегами и описанием
- **Детальная страница проекта:** полная информация, цена, релевантность
- **Фильтрация и сортировка:** поиск по названию (опционально)

### 👥 Система подписчиков (новое в 2026)
- **Подписка на проект:** выбор роли из списка, отправка заявки на сервер
- **Управление ролью:** возможность сменить роль после подписки
- **Отписка:** выход из проекта с обновлением списка участников
- **Список команды:** таблица подписчиков с сортировкой по ролям/именам
- **Индикация:** подсветка текущего пользователя («Вы»)
- **Статистика:** отображение количества подписчиков

### 🎯 Специальные страницы
- **«Совместное выполнение»:** публичная страница с приветственным текстом
  - Для гостей: интерактивный баннер с призывом войти
  - Для авторизованных: список проектов с формами быстрой подписки

## 🌐 Работа с API

Приложение взаимодействует с реальным бэкендом организаторов: `https://webcomp.bsu.ru`

| Действие | Метод | Эндпоинт | Описание |
|----------|-------|----------|----------|
| Логин | `POST` | `/sanctum/token` | Получение токена доступа |
| Проекты | `GET` | `/api/allProjects` | Список всех проектов |
| Роли | `GET` | `/api/2026/qualy/getallroles` | Доступные роли для подписки |
| Подписаться | `POST` | `/api/2026/qualy/addsub` | Заявка на участие (требуется токен) |
| Сменить роль | `POST` | `/api/2026/qualy/editsub` | Обновление роли (требуется токен) |
| Отписаться | `GET` | `/api/2026/qualy/deletesub/{id}` | Выход из проекта (требуется токен) |
| Команда | `GET` | `/api/2026/qualy/projectsubs/{id}` | Список участников проекта |

**Обработка ошибок API** (401, 403, 404, 409, 422) реализована на уровне сервиса с информированием пользователя.
